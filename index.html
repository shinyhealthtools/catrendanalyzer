About the Cochran–Armitage Trend Test

The Cochran–Armitage Trend Test (CATT) is a statistical method for detecting trends in proportions across ordered categorical variables. It is especially valuable when the explanatory variable has a natural ordering and the outcome is binary.

Commonly applied in:

  • Clinical Trials: Testing if treatment response increases with dosage levels
  • Epidemiology: Analyzing disease incidence across ordered age groups
  • Public Health: Examining vaccination rates across socioeconomic strata
  • Genetics: Testing genotype–phenotype associations (dose-response modeling)
  • Market Research: Analyzing preference trends across product variants
Z = Σ yⱼ(xⱼ − x̄) / √[ p̄(1−p̄) · Σ nⱼ(xⱼ − x̄)² ]

Key assumptions: ordered categories, independent observations, binary outcome, sufficient sample size for normal approximation, non-zero marginal totals.

Mathematical Foundation

The test statistic is computed as:

Z = (Σ yⱼ(xⱼ − x̄)) / √[ p̄(1−p̄) · Σ nⱼ(xⱼ − x̄)² ]
  • xⱼ — Scores for ordered groups (default: 1, 2, 3, …)
  • yⱼ — Success counts in each group
  • nⱼ — Total counts in each group
  • x̄ — Weighted mean: Σ(nⱼ·xⱼ)/N
  • p̄ — Overall proportion of successes: Σyⱼ/N
  • N — Grand total: Σnⱼ

Signed Z-statistic: CATrend Analyzer reports a signed Z value. A positive Z indicates an increasing trend; a negative Z indicates a decreasing trend. The p-value is always two-sided.

Continuity correction:

Z = (|Σ yⱼ(xⱼ − x̄)| − 0.5) / √[ p̄(1−p̄) · Σ nⱼ(xⱼ − x̄)² ]

Step-by-step algorithm:

1. Assign scores xⱼ = 1, 2, …, k 2. Compute column totals nⱼ and success counts yⱼ 3. Compute weighted mean x̄ = Σ(nⱼ·xⱼ) / N 4. Compute overall proportion p̄ = Σyⱼ / N 5. Numerator = Σ yⱼ(xⱼ − x̄) 6. Variance = p̄(1−p̄) · Σ nⱼ(xⱼ − x̄)² 7. Z = (|Numerator| − correction) / √Variance, sign from Numerator 8. p-value = 2 · (1 − Φ(|Z|)) [two-sided]
Interpretation Guidelines
  • p < 0.001 — Highly significant trend
  • p < 0.01 — Very significant trend
  • p < 0.05 — Statistically significant trend
  • p ≥ 0.05 — No significant trend detected
Statistical significance does not imply clinical importance. Always consider effect size and practical context when interpreting results.
Frequently Asked Questions
What is the minimum sample size required?
There is no strict minimum. The test relies on a normal approximation, so it works best when expected counts per cell are not too small. A common guideline is ≥ 20 observations per group. For smaller samples, consider exact methods.
Can I use this test for more than two outcome categories?
No. CATT requires a binary outcome. For ordinal outcomes with multiple categories, consider the Jonckheere–Terpstra test or linear-by-linear association test.
What if my groups are not equally spaced?
CATrend Analyzer assigns default equally-spaced scores (1, 2, 3, …). If your groups are unequally spaced, scores should be adjusted manually — currently not supported in the UI, so calculate manually or contact the developer.
How do I cite CATrend Analyzer?
Use the citation provided in the results panel after running an analysis. The citation follows the APA format and includes the retrieval URL.
How does this differ from a chi-square test?
The chi-square test of association detects any association between variables. CATT specifically tests for a linear trend in proportions across ordered groups, making it more powerful when a monotone trend is expected.
Troubleshooting
Zero cells: The test may still run, but interpret results cautiously. Consider aggregating groups or using exact methods.
Small samples (N < 20): Normal approximation may be invalid. Enable continuity correction or consult a statistician.
Missing data: All cells must be filled with non-negative integers. Handle missing data before entering values.

// Core Cochran-Armitage computation (simplified) function performCATT(data, continuityCorrection) { const scores = data.cols.map((_,i) => i + 1); const N = data.flat().reduce((a,b) => a+b, 0); const nj = scores.map((_,j) => data.reduce((s,r) => s+r[j], 0)); const yj = data[0]; // first row = success const xbar = nj.reduce((s,n,j) => s + n*scores[j], 0) / N; const pbar = yj.reduce((a,b) => a+b, 0) / N; const num = yj.reduce((s,y,j) => s + y*(scores[j]-xbar), 0); const den = Math.sqrt(pbar*(1-pbar)*nj.reduce((s,n,j) => s + n*(scores[j]-xbar)**2, 0)); const cc = continuityCorrection ? 0.5 : 0; const Z = (Math.abs(num)-cc)/den * Math.sign(num); return Z; }
